# Question Of the Assignments


1) What is the difference between null and undefined?
> কোন ভেরিয়েবল ডিক্লেয়ার করার পরে যেটির ভ্যালু সেট না করলে সেটা undefined হিসেবে থাকে । আর নিজে থেকে যদি ফাকা  রাখার জন্যে NULL দেওয়া হয় সেক্ষেত্রে NULL থাকবে 


2) What is the use of the map() function in JavaScript? How is it different from forEach()?
> কোন অ্যারের মধ্যে নতুন ডাটা সেট করার জন্যে অথবা অন্য কোন চেঞ্জ আনার জণ্যে map() ব্যবহার করা হয় 
>
>কোন অ্যারের মধ্যে থেকে একটি বা প্রতিটি অব্জেক্ট বের করার জন্যে forEach() ব্যবহার করা হয় 


3) What is the difference between == and ===?
> == এর ক্ষেত্রে শুধু ভ্যালো বা ডাটা মিলছে কিনা দেখা হয় ।
>
>===  এর ক্ষেত্রে ভ্যালু এর পাশপাশি ডাটা টাইপ ও যাচাই করা হয় 


4) What is the significance of async/await in fetching API data?
> api call করার জন্যে async/await ব্যবহার করা হয় যাতে অন্য কোড api লোড হওয়া পর্যন্ত অপেক্ষা করতে না হয় বরং নিজের মত চলতে পারে ।

5) Explain the concept of Scope in JavaScript (Global, Function, Block).
> Global scope এর ক্ষেত্রে ভেরিয়েবল কোড এর যেকোন জায়গায় ব্যবহার করা যায় 
>
>Function এবং block এর ক্ষেত্রে ভেরিয়েবল কোড শুধু ফাংশন বা ব্লকের ভিতরের ব্যবহার করা যায় । বাইরে সম্ভব না ।
